<p>所以各类中共所谓的精英人士，在华府和纽约四处奔走，寻找能与川普联系的人脉服软，但是都吃了闭门羹。甚至还有拜登不会为习近平给未来政府传话的新闻出来，这意味着习这边，也尝试联络拜登寻求帮助呗，然而中共仍然没有放弃，据说已经找了几十个美国政商两界的高层人士了。</p>
<p>Q1：我们先说美国这边。川普的新内阁争议不断，比如前国会议员盖兹 ( Matt Gaetz ) 、前夏威夷州众议员加巴德 ( Tulsi Gabbard ) 、前《福斯新闻》新闻主持人赫格塞斯 ( Pete Hegseth ) 等人，有的被质疑专业能力不够？他们专业能力真的不够吗？</p>
<p>Q2：我们不知道川普团队为什么要让中共吃闭门羹。但是，前一阵，美国政府发现，有一个为中共情报工作的、叫“盐台风”的黑客组织，侵入了美国电信系统，专门针对美国高级政府官员，其中攻击的名单中有川普，万斯，川普儿子等，万斯自己也爆出他的手机被黑客黑了。中共这样做，目的是什么？没想过被爆出的后果吗？</p>
<p>Q3 ：中共最近对美国在频频示好。看来中共是真怕了呀。驻美国大使<ahref="https://github.com/1992513/ntdtv/blob/master/gb/focus/谢锋">谢锋</a>甚至说，要像爱护母亲河一样维护中美关系。中南海也在联络那些在中国有利益的商人，比如是<ahref="https://github.com/1992513/ntdtv/blob/master/gb/focus/马斯克">马斯克</a>，希望他们能够帮助川普说话。中共这样做，有用吗？<ahref="https://github.com/1992513/ntdtv/blob/master/gb/focus/谢锋">谢锋</a>可能是不知道啊，母亲河被破坏得好像都断流了啊。</p>
<p>Q4：英国《金融时报》还透露，中共还出动了川普执政期间驻美的前大使崔天凯，但是八名知情人士表示，这些努力都无功而返。要知道崔天凯号称拥有中共官员中最好的华盛顿人脉。这一系列的操作，不知道有没有把川普团队看蒙，据说中南海现在特别焦虑，拚命想知道“川普到底想要什么”，大家说，川普真正想要什么？</p>
