<tr><td VALIGN=TOP width="626"><h2 align=center>彭佳慧10月再登小巨蛋 最期待刘德华任嘉宾</h2>
<img width="600" src="https://i.ntdtv.com/assets/uploads/2022/07/0-crop-800x450.jpg" />
<h6>彭佳慧资料照（黄宗茂／大纪元）</h6>
<hr>
	<p>【新唐人北京时间2022年07月14日讯】<ahref="https://github.com/19920513/ntdtv/blob/master/gb/彭佳慧.md#1">彭佳慧</a>睽违五年，昨（13）日在记者会上宣布将于10月1日在<ahref="https://github.com/19920513/ntdtv/blob/master/gb/台北小巨蛋.md#1">台北小巨蛋</a>举行“彭佳慧2022 [怎样 Julia / Run Wild ] 演唱会”。为了这次开唱，她铆足全力准备，除了运动与保养，也因为确诊过后更积极练唱当作复健。</p>
<p><figure id="attachment_103479165" style="width: 400px" class="wp-caption aligncenter"><img class="size-full wp-image-103479165" src="https://i.ntdtv.com/assets/uploads/2022/07/1-111.jpg" alt="" width="400" b="267" /><figcaption class="wp-caption-text"><ahref="https://github.com/19920513/ntdtv/blob/master/gb/彭佳慧.md#1">彭佳慧</a>“2022 [怎样 Julia / Run Wild ] 演唱会”于7月13日在台北举行记者会。（黄宗茂／大纪元）</figcaption></figure>她对此表示：“我们可以用唱歌来做一种复健，因为英国好像有做一些研究，唱歌不但可以舒张你的肺部，唱完歌以后你的情绪也有某一种程度的宣泄，身心都有治愈到，所以我觉得唱歌这件事满好的。”</p>
<p>彭佳慧透露自己染疫后的身体状况有一个，就是关节会常常比较容易痛，手跟脚都有，以前不会这样，不过她没有去看医生，何况并不会影响唱歌，但是她不否认现在高跟鞋穿的时间比以前稍短一点。</p>
<p>之前一度担心喉咙受影响，彭佳慧也坦承实际上“偶尔真的会有一点胸闷”，但也觉得这件事情不用一直特别去想它，毕竟确诊过后到现在，也没有很久的时间。</p>
<p><figure id="attachment_103479164" style="width: 400px" class="wp-caption aligncenter"><img class="wp-image-103479164" src="https://i.ntdtv.com/assets/uploads/2022/07/2-crop-3-600x338.jpg" alt="" width="400" b="225" /><figcaption class="wp-caption-text">彭佳慧“2022 [怎样 Julia / Run Wild ] 演唱会”于7月13日在台北举行记者会。（黄宗茂／大纪元）</figcaption></figure>昨日记者会也特别整理彭佳慧出道25年来的经典歌曲，从1996年《旧梦》、1999年《走在红毯那一天》、2009年《无法割舍》、2015年《大龄女子》到2021年的《太难唱了》等，每首歌都承载着你我的青春岁月。</p>
<p>彭佳慧感性地说：“其实原本是要庆祝25周年，不小心就变成26年了（笑），希望这个数字能为跟我一样爱音乐的朋友带来幸运，毕竟一件事情要做26年，然后越做越好 也是不简单的事。）</p>
<p>被问及有无演唱会嘉宾梦幻名单，彭佳慧直言：“讲出来就没有意思啦！刘德华！因为我永远记得我有一年演唱会，BMG演唱会上我跟华仔手牵手唱《愿牵你的手》，他专辑里面的歌。”</p>
<p>她回忆当时自己还是新人，那时候的她没有表现好，“如果还有机会再让我还有一次机会，应该不那么紧张，表现得更好。”</p>
<p><figure id="attachment_103479166" style="width: 400px" class="wp-caption aligncenter"><img class="size-full wp-image-103479166" src="https://i.ntdtv.com/assets/uploads/2022/07/3-20.jpg" alt="" width="400" b="600" /><figcaption class="wp-caption-text">彭佳慧“2022 [怎样 Julia / Run Wild ] 演唱会”于7月13日在台北举行记者会。（黄宗茂／大纪元）</figcaption></figure>这次开唱，彭佳慧希望完整传递每首歌的情感和故事，并说：“歌单上的量在这个阶段排出来已经超时了，但我这次坚持的是让每一首歌都能完整的被表现，也会尽力让歌迷们能搭上最后一班捷运回家。”</p>
<p>（转自大纪元/责任编辑：叶萍）</p>
